🚀 QUICK START - 3 Steps

1. INSTALL
   pip install -r requirements.txt

2. LAUNCH
   streamlit run app.py
   
3. USE
   - Upload CSV (must have NL, X1, Y1, pDBH_RF, Z_ft, pStV_RF)
   - Select strategies
   - Click "Run Analysis"
   - Wait 10-30 min
   - View results
